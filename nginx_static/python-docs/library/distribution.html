软件打包和分发¶

这些库可帮助你发布和安装 Python 软件。 虽然这些模块设计为与`Python 包索引 <https://pypi.org>`__结合使用，但它们也可以与本地索引服务器一起使用，或者根本不使用任何索引服务器。

  • distutils --- 构建和安装 Python 模块
  • ensurepip --- Bootstrapping the pip installer
